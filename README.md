# Multilingual AI Civic Grievance Assistant

## JanVoice AI 2.0 — AI Civic Assistant

This project is designed to help citizens:

- Identify the right authority for a civic issue
- Generate complaint text in multiple formats
- Recommend next actions and escalation paths
- Suggest the best grievance portal and department
- Provide local language output for Hindi and Telugu

## Features

- Issue classification and authority mapping
- Formal letter, email, WhatsApp, and X/Twitter drafts
- Severity estimation and escalation guidance
- Andhra Pradesh-focused sample authority database
- Streamlit UI with analysis workflow
